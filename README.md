# Zed Editor Settings Backup & Restore

## 📌 How to Copy and Restore Your Zed Settings

If you want to **backup or transfer** your Zed Editor settings to another system, follow these simple steps:

### **🔹 Method 1: Using Zed Editor (Recommended)**
1. Open **Zed Editor**.
2. Navigate to **Settings**.
3. Copy and paste the current settings to a safe location.
4. On the new system, paste them into Zed's settings panel.

### **🔹 Method 2: Manually Copy Configuration Files**
1. Open a terminal and navigate to the Zed configuration folder:
   ```bash
   cd ~/.config/zed
   ```
2. Copy the following two files from the old system to this directory:
   - `settings.json`
   - `keybindings.json`
3. Restart **Zed Editor** to apply the changes.

### ✅ **Enjoy Your Custom Settings!**
After restarting Zed, your settings should be restored, and you can continue coding with your preferred setup.

🚀 Happy Coding! If you face any issues, feel free to tweak the settings manually in `~/.config/zed/`.
